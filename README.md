# Semaforo

Sistema interativo de controle de semáforos.
Conecte-se a um semáforo usando seu ID e controle as luzes de sinalização remotamente. O sistema simula o gerenciamento de semáforos em tempo real com feedback visual e monitoramento de status.

## Teste

Clone o repositorio.

```bash
git clone https://github.com/ebbendje/Sem-foro.git
```

Abra `index.html` em um navegador

---

## Features

Here're some of the project's best features:

-   Conectar a um ip device
-   Executando automaticamente depois de conectado
-   Mudar o sinal do semafor de veiculos
-   Ver log do que esta acontecendo por tras
-   Monitoramento da hora da mudanca dos sinais
